# Motif Atlası

Anadolu tekstil motifleri interaktif web deneyimi.

## Deploy

### Vercel
Vercel.com → New Project → klasörü upload et → Deploy

### GitHub Pages  
1. `git init && git add . && git commit -m "init"`
2. GitHub'a push et
3. Settings → Pages → main branch / root

## Klasör yapısı
```
motif-atlasi/
├── index.html
├── css/style.css
├── js/script.js
├── assets/
│   ├── images/   ← tüm görseller
│   └── videos/   ← mp4 videolar
└── vercel.json
```
